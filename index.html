    Erfahrungsbericht:

    Das Erstellen des Layouts mit Hilfe von CSS hat einwandfrei funktioniert. Für kompliziertere Seiten, welche nicht im Rahmen einer Klausurarbeit erstellt wurden, würde ich in Zukunft jedoch das Frontend Framework "Bootstrap" verwenden.
    Bei dem Erstellen der Weather-Api stellte sich heraus, dass die vorgeschlagene "5 Day / 3 Hour Forecast"-Funktion von "https://openweathermap.org/api" nicht kostenlos verfügbar ist. Daher habe ich mich für den aktuellen Wetterbericht entschieden. Um dennoch zu demonstrieren, dass ich in der Lage bin mehrere, voneinander teilweise unabhängige Dinge aus einer API einzubauen, besitzt die Datei "bunnys.js" eine auskommentierte Version zur Einbindung verschiedener Dinge aus einer API.
    Das Erstellen der Instagram-API ist mir zunächst sehr schwer gefallen, da ich mir eigentlich über diesen Weg die möglichen Zusatzpunkte holen wollte. In diesem Zusammenhang wollte ich über die "Instagram Graph API" gehen. Diese erfordert jedoch neben einem Facebook-Developer-Account auch ein verifiziertes Unternehmen. Zu dem Thema Instagram-API lässt sich jedoch zusammenfassend sagen, dass es extrem aufwendig ist und für die meisten Anwendungsfälle auch eine unkomplizierte Methode über Instagram funktionieren würde. Ich würde daher im Normalfall über die Option "Einbetten" gehen, welche Instagram selber zu jedem Beitrag zur Verfügung stellt. In diesem Zusammenhang lässt sich noch sagen, dass ich das Logo der Webseite selber erstellen musste, da die Seite zwangsweise veröffentlicht werden musste (zum Einbinden der Instagram-API). Aufrund der Veröffentlichung habe ich auch alle meine Quellen im Impressum angegeben.
    Das Verwenden der JavaScript-Bibliothek "jQuery" hat einwandfrei funktioniert. Ich würde diese jederzeit wieder verwenden.
    Im Nachhinein ist mir aufgefallen, dass ich mit einer besser unterteilten Ordnerstruktur hätte arbeiten sollen.
    Ich habe das Datenflussdiagramm vereinfacht gezeichnet. Da es so übersichtlicher ist, habe ich nur alle Möglichkeiten von der Startseite ausgehend dargestellt. Natürlich kann ich von jeder der Seiten zu allen anderen Seiten wechseln.
